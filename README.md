使用说明
---

三叶草最大可设置为 `999999999`

抽奖券最大可设置为 `999`

每次可将上次退出游戏时间提前 `3` 小时，可连续多次点击

可以将明信片无损导出到 `/sdcard/Pictures/Tabikaeru`
